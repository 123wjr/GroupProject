# GroupProject
The group 6 project repo for NUS scale youth program

# 项目要求

总体提交要求

* 形式： 以小组为单位完成。
* 提交物： 所有文件必须打包成一个名为 
"Group-X-Project.zip" 的压缩文件（请将 X 替换为你的小组编号）。
* 截止日期： 在截止日期前提交至 Canvas 的指定文件夹。

压缩包内必须包含以下文件：

1. PowerPoint 幻灯片： 名为 
"Group-X-Project.pptx"（最多10张幻灯片）。
2. 工作代码文件： 名为 
"Group-X-Project.ipynb" 的 Jupyter Notebook 文件。
3. 代码打印预览： 名为 
"Group-X-Project.pdf"，是 ipynb 文件的打印预览（包含所有运行结果）。
4. 完成的 Word 文档： 名为 
"Group-X-Project.docx"（包含所有问题和结果）。
5. 数据文件： 使用的原始数据文件（csv 或 excel 格式）。

具体任务清单

第一部分： 准备与数据收集

1. 阅读作业： 阅读指定文章《An Introduction to Population Growth》。
2. 新加坡数据： 从新加坡统计局（SingStat）网站下载新加坡 1950 年至 2024 年的人口数据。
3. 本国数据： 从世界银行（World Bank）网站下载你所在国家 1950 年至 2024 年的人口数据。

第二部分： 新加坡人口回归分析（需在报告和代码中完成）

1. 绘图： 绘制新加坡总人口与年份（1950-2024）的关系图。
2. 建模：
   * 使用 2020 年及之前的数据作为训练数据。
   * 使用线性回归方法构建一个未来人口预测模型。
3. 性能评估：
   * a 给出最佳拟合直线的斜率和截距。在数据图上绘制出这条最佳拟合直线。
   * b 计算该模型在训练数据上的 R² 系数 和 均方误差（MSE）。
   * c 使用 2021 年及之后的年份作为测试数据，预测这些年份的人口。
   * d 计算模型在测试数据上的 MSE。
4. 预测与评估：
   * 预测新加坡在 2025年、2030年 和 2050年 的人口。
   * 讨论这些预测结果是否合理，并解释原因。
5. 模式思考： 你认为新加坡的人口增长未来会呈现什么模式？
6. 创造性改进： 思考并描述你可以如何改进未来人口的预测估计。

第三部分： 本国人口回归分析

* 重复第二部分的全部任务，但将数据对象改为你在第一部分中下载的你所在国家的人口数据。

第四部分： 项目演示（需制作PPT）

准备一个不超过10分钟的演示，内容需涵盖以下要点：

1. 引言： 阐述为什么人口预测是一个值得研究的重要问题。
2. 问题描述： 清晰说明你要解决的问题及其背景。
3. 实验设置： 描述你所做的任何假设和实验设置。
4. 结果展示： 以清晰易懂的方式可视化你的结果。
5. 结果分析： 从结果中获得了哪些洞察？关键反思是什么？
6. 反思与结论： 本项目的主要结论是什么？你学到了什么？

第五部分： 附录（知识参考）

* 理解并应用 R² 系数 的计算方法和意义。
